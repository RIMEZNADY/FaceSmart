# FaceSmart

**Developed by:** Znady Rime & Chakra Hossam  

##  Project Overview
**FaceSmart** is an attendance tracking system based on **facial recognition**.  
It provides a **complete solution** for automatic attendance management with a **web interface** and an **interactive dashboard** for employees.

##  Features
- Real-time facial recognition for attendance logging  
- Secure and automated attendance management  
- Web interface for easy monitoring  
- Interactive dashboard with employee insights  

##  Technologies Used
- Python (Face Recognition & Backend)
- OpenCV (Facial Detection)
- Tkinter (Frontend)
- Database (MySQL )
